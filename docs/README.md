# ESP-32 GAMEPAD (Bluetooth)

a ESP-32 gamepad made purely in rust with esp-idf (with std)
(this project is not bare metal rust)

this repo contains code files of a custom gamepad/game controller which not just can be used to 
play games on some device but this has it's own games

on startup you get a menu on a OLED Display wether you want to connect with bluetooth , play games on it, or see settings

just like any other gamepad you get 4 action button , 4 movement buttons or a DPAD, l1-l2 , r1-r2, r3-l3,
start , select and a unique back button, 
back button - this button is not known to bluetooth devices and is only for nevigating in the gamepad
                this can be used to exit from the current action such as bluetooth or any game


it has built in games like snake , tetris and many more (planning, yet to be developed),
and since this is open source you are freee to develop your own game and play
