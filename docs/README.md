This project is a wireless game pad made using esp32, 

buttons available are- 
up,down,left,right.
x,y,a,b.
l1,l2,l3
r1,r2,r3.
2 joysticks

and the most unique feature - An OLED display;

this gamepad can be used to play games like snake which are developed by myself especially for this esp + OLED combo in RUST 

on boot a loading screen(for feel)
and a menu is displayed on the screen asking user to use bluetooth or play offline on chip games , user can nevigate in menus and choose

there's also a game known as debug and it exists just to ensure if all the buttons are working or not , it only shows state of all buttons
